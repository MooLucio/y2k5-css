> "The visual style of the web in 2005 is back, baby!"

`y2k5-css` is a lightweight CSS framework that meticulously recreates the authentic aesthetic of the web around 2005, capturing the essence of the Windows XP era, early Web 2.0, and the peak of personal websites.

Perfect for retro web projects, nostalgic throwbacks, or simply embracing the glorious visual chaos of the early 2000s.

## âœ¨ Key Features (The Nostalgia Trip)

- **Windows XP Blue Gradients:** The iconic blue gradient (`var(--primary-color)`) is the default, used in panel headers, tables, and primary buttons.
- **Grid Layouts via Tables:** Authentic early-2000s layout methodology using `display: table` for the grid (`.row` and `.col` classes).
- **Verdana 11px:** The default typeface, paired with the classic small font size (`11px`), ensuring an unmistakable mid-2000s feel.
- **Solid Borders & Sharp Corners:** Everything features crisp, 1px solid borders (`#666` or contextual colors).
- **The Yellow Focus:** Form fields get the famous **Windows XP yellow focus** on activation (`:focus` state).
- **Four Quick Themes:** Easily switch the primary color palette with body classes like `.theme-pink`, `.theme-lime`, `.theme-orange`, and `.theme-purple`.
